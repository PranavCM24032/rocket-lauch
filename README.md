# Rocket Games (Pygame)

This repository contains two Python/Pygame rocket game variants:

- `game.py`: base version
- `space.py`: enhanced version (pickups, boss asteroids, combo, high score, effects)

## Quick Start

```bash
pip install -r requirements.txt
python game.py
```

Or run:

```bash
python space.py
```

## Project Files

- `index.html`: GitHub landing page linking both game files
- `game.py`: original game build
- `space.py`: expanded game build
- `rocket_highscore.json`: local high score save file for `space.py`
- `assets/`: runtime/downloaded assets (e.g., pickup icons)

## Notes for GitHub

- `index.html` is intended as a repository hub page.
- GitHub Pages cannot execute Pygame directly in the browser.
- To play the games, run them locally with Python.
